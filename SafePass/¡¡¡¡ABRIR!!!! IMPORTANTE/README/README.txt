[ + ] UTILIDAD APLICACIÓN : La aplicación sirve para guardar todas las contraseñas que uses 
      en las aplicaciones / páginas web y evites todas las veces que le das a "he olvidado mi contraseña". 

[ + ] CÓMO FUNCIONA : Primero se ejecuta una ventana (GUI) en la cual si le das al título de la aplicación se abre 
      una nueva ventana. En esta nueva ventana puedes añadir tus contraseñas indicando para que usas esas contraseñas. 
      Si sientes que tu contraseña no es muy segura, tienes la opción de generar una contraseña aleatoria que sí sea segura. 
      Si elijes esta última opción, se abrirá una tercera ventana en la cual te muestra la contraseña generada y tienes 2
      botones : "Copiar" y "Cancelar". El de copiar te copiará la contraseña para que la pongas en el apartado de contraseñas. 
      El botón de cancelar cierra esa ventana sin copiar la contraseña ni hacer nada. Ahora tienes un ejemplo de lo que 
      debes introducir para guardar la contraseña de por ejemplo tu correo electrónico (Gmail).
      > > Ejemplo :                            	 > > Ejemplo : 
        > Aplicación / web : Gmail                	   > Aplicación / web : YouTube
        > Contraseña : password_123                > Contraseña : tengo_1m_subs_xd_999

[ + ] PRIVACIDAD : Poner tus contraseñas en una aplicación que no conoces, puede dar mal rollo, e incluso no te puede gustar
      y puedes estar desconfiado. SafePass no roba información, ni envía tus contraseñas a un servidor para que otra gente 
      la sepa. Símplemente SafePass quiere ayudarte a no tener que darle nunca más a "he olvidado mi contraseña" y que 
      tengas tus contraseñas mucho más accessibles.
